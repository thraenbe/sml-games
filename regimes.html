Statistical Machine Learning · Exercise 2 · Linear Regression in Different Regimes

One best fit, infinitely many perfect fits, and a dial between

Whether least squares has a single answer is decided by counting: n data points against d parameters, refereed by the rank of X. This page walks parts (a)–(i): ordinary least squares when data outnumbers parameters, the minimum-norm interpolant when parameters outnumber data, and ridge regression — a dial that buys uniqueness everywhere.

the solution set — every w that fits the perpendicular step — minimum-norm choice ridge — the λ dial

Setup · the regime mapTwo regimes, one referee: rank

Everything in this exercise hangs on one comparison. If n ≥ d and rank(X) = d, the columns of X are linearly independent but cannot span ℝn — the data generally cannot be fit exactly, and least squares picks the best compromise. If d > n and rank(X) = n, the columns span all of ℝn — every data vector can be fit exactly, in infinitely many ways, and the problem becomes choosing among perfect fits.

Parts (a) & (b) · n ≥ dOverdetermined: the norm trick makes OLS unique

Why is XTX invertible? The whole proof is one identity, worth naming because this exercise uses it three times — call it the norm trick:

vTXTX v = ‖Xv‖²

Suppose XTXv = 0. Multiply by vT: then ‖Xv‖² = 0, so Xv = 0. But rank(X) = d means the columns are independent, so Xv = 0 forces v = 0. A square matrix whose null space is {0} is invertible. Combined with the normal equations XTXw = XTY from Exercise 1, invertibility hands us the solution — and because an invertible system has exactly one solution, it is the minimizer:

ŵ = (XTX)−1XTY

Parts (c) & (d) · d > nUnderdetermined: everything fits

Now flip the regime. With rank(X) = n, the d columns of X span all of ℝn, so every Y is a linear combination of columns: some w satisfies Xw = Y exactly. Zero loss is always achievable — the model can reproduce any data, noise included. The price appears immediately: if Xw = Y and Xv = 0 with v ≠ 0, then w + v also fits perfectly. With d > n the null space of X is nontrivial, so there are infinitely many perfect fits.

Part (d) is the norm trick, second appearance, with the roles of rows and columns swapped: if XXTu = 0 then uTXXTu = ‖XTu‖² = 0, so XTu = 0 — meaning u is perpendicular to every column. Columns that span ℝn leave no room for such a u except 0, so XXT (an n × n matrix now) is invertible.

Parts (e) & (f) · picking oneAmong all perfect fits, take the shortest

Invertibility of XXT lets us write down a distinguished interpolant, the minimum-norm interpolant:

ŵmn = XT(XXT)−1Y

Part (e) is a one-line check: X ŵmn = XXT(XXT)−1Y = Y — the inverse cancels and it really does fit the data exactly. Part (f) says it is the unique shortest one, and the argument is Exercise 1's triangle all over again — same proof, new plane. There the residual stood perpendicular to image(X) in data space; here ŵmn stands perpendicular to the null space of X in weight space.

(f) step 1 · Xv = 0

Let w be any interpolant and set v = w − ŵmn. Both fit perfectly, so Xv = Xw − X ŵmn = Y − Y = 0: the difference of two perfect fits lies in the null space of X — the blue direction along the line in figure 1.

(f) step 2 · ŵmn ⊥ v, then Pythagoras

ŵmnTv = (XT(XXT)−1Y)Tv = YT(XXT)−1Xv = 0. So ‖w‖² = ‖ŵmn‖² + ‖v‖², minimized uniquely at v = 0. The perpendicular wins — again.

Part (g) · ridge as disguised OLSRidge: change the question, not the answer's form

Minimum-norm is one way to break the tie among perfect fits. Ridge regression instead changes the objective, penalizing large coefficients:

minimize   ‖Xw − Y‖² + λ‖w‖²,   λ > 0

The trick of part (g): this is not a new kind of problem at all. Stack d fake observations under the real ones, each demanding that one coordinate of w be 0 with weight √λ:

X̃ = X√λ Id,    Ỹ = Y0,   so   ‖X̃w − Ỹ‖² = ‖Xw − Y‖² + λ‖w‖²

The augmented residual stacks the true residual on top of √λ w, and squared norms add block by block. Ridge is ordinary least squares — on a taller, better-behaved system.

Parts (h) & (i) · uniqueness for freeThe augmented system is always overdetermined

(h) X̃ always has independent columns

If X̃v = 0, the bottom block alone reads √λ v = 0, and λ > 0 forces v = 0 — regardless of the rank of X. So part (a)'s norm trick, third appearance, applies to X̃: the matrix X̃TX̃ is invertible and ridge has exactly one minimizer in both regimes.

(i) normal equations, augmented

Multiplying out the blocks: X̃TX̃ = XTX + λId and X̃TỸ = XTY. The normal equations for the tall system are therefore (XTX + λId)w = XTY.

ŵλ = (XTX + λId)−1XTY

Adding λI is exactly what makes the matrix invertible when XTX alone is not — the penalty term props up the directions the data leaves unconstrained.

SummaryThree formulas, one pattern

regimeconditionsolutionunique?
OLS n ≥ d, rank d ŵ = (XTX)−1XTY yes — XTX invertible
min-norm d > n, rank n ŵmn = XT(XXT)−1Y interpolants: no — shortest one: yes
ridge any X, λ > 0 ŵλ = (XTX + λI)−1XTY yes — always

The one-line summary. Uniqueness in least squares is never an accident — it is always some matrix being invertible, and all three invertibility proofs here are the same norm trick vTM v = ‖·‖². When the data can't provide invertibility (d > n), you either add a rule (shortest interpolant) or add a penalty (ridge) — and the two meet: ŵλ → ŵmn as λ → 0.
